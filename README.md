# Charitable Donations System

## 📌 About Project
This project is a web-based charitable donation system built using Laravel.  
It allows users to donate to different cases such as:
- Zakat
- Charity
- Social cases
- Blood money (Diyya)

## ⚙️ Features
- User registration & login
- Donation cases management
- Wallet system
- Transfer donations between users
- Admin dashboard

## 🛠️ Technologies Used
- Laravel
- PHP
- MySQL
- Tailwind css
- Alpine.js
